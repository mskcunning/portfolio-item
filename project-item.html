Project Description

Fully plan, deisgn, and implement a network topology in Cisco Packet Tracer. This includes creating an IP addressing scheme, applying device configurations, and producing the required documentation.

Tasks Completed

  • Create a functional and efficient IP addressing scheme
  • Determine and implement the necessary networking devices
  • Apply the proper configurations to all hosts and networking devices in the topology
  • Produce the required documentation including a cover letter, address table, and list of device configurations

This project entailed planning, designing, and implementing a complete network topology using Cisco Packet Tracer. I first created a full IP address topology and added the necessary network devices to the topology. I then configured all devices in the topology including naming, security, and IP configurations. Finally, I created the required documentation for the network including a cover letter, an IP addressing table and a full description of all configurations performed.
